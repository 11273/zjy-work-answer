## 📖 使用指南

### 🎯 启动程序

```bash
python StartWork.py
```

### 🔐 选择登录方式

程序启动后会显示登录选择界面：

```
============================================================
🚀 ZJY助手登录服务
============================================================
请选择登录方式：
1. 用户名密码登录（传统方式）
2. OAuth浏览器登录（推荐）
============================================================
```

#### 方式 1️⃣：OAuth 登录（推荐）

1. **选择选项 2**
2. **自动打开浏览器**：程序会自动打开智慧职教登录页面
3. **完成登录**：在浏览器中正常登录您的账号
4. **自动回调**：登录成功后页面会显示成功信息并自动处理
5. **继续使用**：返回程序界面继续操作

#### 方式 2️⃣：传统登录

1. **选择选项 1**
2. **输入账号**：输入您的用户名
3. **输入密码**：输入您的密码
4. **登录验证**：程序自动处理登录流程

### 🔄 自动学习流程

程序会自动执行以下步骤：

1. **📋 获取课程列表**：自动检测账号下的所有课程
2. **🎯 课程筛选**：根据设置跳过指定课程
3. **▶️ 开始学习**：自动播放视频和完成学习任务
4. **📊 进度显示**：实时显示学习进度和状态
5. **✅ 完成提示**：所有课程完成后显示结果
